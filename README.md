# 三组扑克牌游戏 - 多人在线对战版

## 🎮 游戏特色

- **可爱风格界面**：粉色渐变背景，卡通字体，精美动画
- **多人在线对战**：支持局域网内多人实时对战
- **智能AI对手**：可以添加电脑对手练习
- **完整游戏规则**：三组游戏（1张牌比大小、十点半、炸金花）
- **特殊计分规则**：黑桃A、十点半、豹子等特殊加分

## 🚀 快速开始

### 方法1：本地游戏（单人+AI）

1. 直接双击打开 `index.html` 文件
2. 输入玩家名称（可选）
3. 添加AI对手
4. 点击"开始游戏"

### 方法2：多人在线对战

#### 启动服务器（主机）

1. 双击运行 `start-server.bat` 文件
2. 等待服务器启动，会显示访问地址
3. 在浏览器中打开 `http://localhost:3000`
4. 点击"创建房间"
5. 将房间ID分享给朋友

#### 加入游戏（朋友）

1. 确保和主机在同一局域网
2. 在浏览器中打开 `http://主机IP:3000`（例如：http://192.168.1.100:3000）
3. 输入房间ID
4. 点击"加入房间"
5. 等待主机开始游戏

## 📋 游戏规则

### 第一组：1张牌比大小
- 基础分：1分
- 特殊加分：持有黑桃A得2分
- 牌面大小：数字优先，相同则按花色（黑桃＞红桃＞梅花＞方块）

### 第二组：十点半
- 基础分：2分
- 特殊加分：十点半得4分（翻倍）
- 点数计算：1-10按牌面，J/Q/K算0.5点
- 超过10.5点爆牌判负

### 第三组：炸金花
- 基础分：3分
- 特殊加分：豹子得6分（翻倍）
- 牌型排序：豹子＞顺金＞金花＞顺子＞对子＞单张
- 特殊规则：235可以压制豹子，但若对方不是豹子则235最小

## 🔧 技术说明

### 文件结构
- `index.html` - 游戏主页面
- `server.js` - Socket.io服务器
- `start-server.bat` - 服务器启动脚本
- `package.json` - 项目配置

### 端口配置
默认端口：3000
如需修改，编辑 `server.js` 文件中的 `PORT` 变量

### 局域网访问
1. 查看主机IP地址（运行 `ipconfig` 命令）
2. 朋友访问 `http://主机IP:3000`
3. 确保防火墙允许3000端口

## 🎯 使用技巧

1. **自动生成玩家名**：不输入名称直接点击会自动生成
2. **取消选择**：点击已选卡牌可放回
3. **牌面隐藏**：只能看到自己的牌
4. **主机控制**：只有主机可以开始游戏

## ⚠️ 注意事项

1. 多人对战需要在同一局域网
2. 确保防火墙允许3000端口
3. 主机必须保持服务器运行
4. 玩家断开会自动从房间移除

## 🐛 常见问题

**Q: 朋友无法访问我的游戏？**
A: 检查防火墙设置，确保3000端口开放

**Q: 玩家名称显示不对？**
A: 刷新页面重新连接

**Q: 游戏卡顿？**
A: 检查网络连接，确保在同一局域网

**Q: 如何停止服务器？**
A: 在服务器窗口按 Ctrl+C
