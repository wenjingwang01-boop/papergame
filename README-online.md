# 三组扑克牌游戏 - 在线版本

## 🎮 游戏特色

- **可爱风格界面**：粉色渐变背景，卡通字体，精美动画
- **多人在线对战**：支持全球范围内的多人实时对战
- **智能AI对手**：可以添加电脑对手练习
- **完整游戏规则**：三组游戏（1张牌比大小、十点半、炸金花）
- **特殊计分规则**：黑桃A、十点半、豹子等特殊加分
- **平局不得分**：平局时双方都不得分

## 🚀 如何使用

### 方法1：在线游戏（推荐）

1. 打开 **https://three-game-client.onrender.com**
2. 输入玩家名称（可选）
3. 选择游戏模式：
   - **本地游戏**：添加AI对手，自己玩
   - **多人对战**：创建房间，邀请朋友一起玩

### 方法2：多人对战

**主机：**
1. 打开游戏网站
2. 点击"创建房间"
3. 将房间ID分享给朋友
4. 等待朋友加入
5. 点击"开始游戏"

**朋友：**
1. 打开游戏网站
2. 输入房间ID
3. 点击"加入房间"
4. 等待主机开始游戏

## 🎯 游戏规则

### 第一组：1张牌比大小
- **基础分**：1分
- **特殊加分**：持有黑桃A得2分
- **牌面大小**：数字优先，相同则按花色（黑桃＞红桃＞梅花＞方块）
- **平局**：双方都不得分

### 第二组：十点半
- **基础分**：2分
- **特殊加分**：十点半得4分（翻倍）
- **点数计算**：1-10按牌面，J/Q/K算0.5点
- **超过10.5点**：爆牌判负
- **平局**：双方都不得分

### 第三组：炸金花
- **基础分**：3分
- **特殊加分**：豹子得6分（翻倍）
- **牌型排序**：豹子＞顺金＞金花＞顺子＞对子＞单张
- **特殊规则**：235可以压制豹子，但若对方不是豹子则235最小
- **平局**：双方都不得分

## 🔧 部署说明

### 客户端部署

1. **创建GitHub仓库**：
   - 创建一个新的GitHub仓库
   - 将 `index.html` 文件上传到仓库

2. **部署到Render.com**：
   - 登录Render.com
   - 点击"New" → "Static Site"
   - 连接你的GitHub仓库
   - 设置构建命令：`echo "Deploying static site"`
   - 设置发布目录：`/`
   - 点击"Deploy"

### 服务器部署

1. **创建GitHub仓库**：
   - 创建一个新的GitHub仓库
   - 将 `server-online.js` 和 `package-online.json` 文件上传到仓库
   - 重命名 `package-online.json` 为 `package.json`

2. **部署到Render.com**：
   - 登录Render.com
   - 点击"New" → "Web Service"
   - 连接你的GitHub仓库
   - 选择Node.js环境
   - 设置构建命令：`npm install`
   - 设置启动命令：`npm start`
   - 点击"Deploy"

3. **更新API地址**：
   - 在 `index.html` 中更新API端点为你的服务器地址

## 🐛 常见问题

**Q: 无法连接服务器？**
A: 检查网络连接，或尝试刷新页面

**Q: 朋友无法加入房间？**
A: 确保房间ID正确，且服务器正常运行

**Q: 游戏卡顿？**
A: 检查网络连接，或尝试使用本地游戏模式

**Q: 如何停止服务器？**
A: 在Render.com控制面板中停止服务
